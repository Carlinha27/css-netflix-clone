## Descrição

Projeto front-end inspirado na tela de seleção de perfis da Netflix.  
O objetivo foi praticar **HTML e CSS**, trabalhando com layout, flexbox, hover e organização visual.

A página exibe perfis de usuários fictícios e um botão para gerenciamento de perfis.

## Preview

Tela de seleção de perfis com:

- Logo centralizado
- Lista de usuários
- Efeitos visuais ao passar o mouse
- Layout centralizado

## Tecnologias Utilizadas

- HTML5
- CSS3
